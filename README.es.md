---

## Ver el sitio en tu PC

```bash
npm install
npm run dev
```

Abre http://localhost:5173

---

## Publicar en GitHub Pages

```bash
npm install
npm run deploy
```

Luego: **Settings → Pages →** rama `gh-pages`, carpeta `/ (root)`.

URL: https://enithv.github.io/Excercises_practice/

---

## Ejecutar los ejercicios en Java

```bash
cd java/src
javac *.java
java Main
```

Elige del **1** al **35**; **0** para salir.
