# Exercises Practice · 35 Ejercicios de práctica

**Repository:** [github.com/EnithV/Excercises_practice](https://github.com/EnithV/Excercises_practice)  
**Live site (GitHub Pages):** https://enithv.github.io/Excercises_practice/

35 classic programming exercises with **in-browser demos**, bilingual UI (EN/ES), and sample solutions in **Java**, **JavaScript**, and **Python**.

[Versión en español](README.es.md)

---

## What's in this repo

| Part | Description |
|------|-------------|
| **Root** (`index.html`, `js/`, `css/`) | Web catalog — 35 exercises, live demos, code tabs |
| **`java/src/`** | Runnable Java (35 classes + `Main.java` menu) |

---

## Run the website locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

---

## Deploy to GitHub Pages

```bash
npm install
npm run deploy
```

Then: **Settings → Pages →** branch `gh-pages`, folder `/ (root)`.

Site URL: https://enithv.github.io/Excercises_practice/

---

## Run Java exercises

```bash
cd java/src
javac *.java
java Main
```

Choose **1–35** in the menu; **0** to exit.
